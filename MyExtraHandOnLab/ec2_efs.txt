<Step 1> Goto Elastic File System (EFS), "Create file system"; enter the name and click Customize button; Goto Networking and edit the Security Group and allow inbound "NSF Port2049"
<Step 2> Back to the EFS, and select the created EFS; then click "Attach" button; copy the mount command (eg. sudo mount -t efs -o tls fs-032056bae231846d9:/ efs)
<Step 3> Create EC2 as usual, then goto EC2 terminal and mount it to the EFS
